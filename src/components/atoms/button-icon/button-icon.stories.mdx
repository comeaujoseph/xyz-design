## Sizing

A button icon supports multiple sizes:

- default
- medium
- small
- x-small
- xx-small

Omit the `size` property for a button icon with the default size.

<Preview withToolbar>
  <Story name="sizing" parameters={{ decorators: [withKnobs] }}>
    <div>
      <ButtonIcon>
        <IconHelp />
      </ButtonIcon>

      </ButtonIcon>

      </ButtonIcon>

## Variations

In xyz design we provide 1 types of button:

- neutral
- dropdown

2 additional properties:

- `disabled`
- `outline`
