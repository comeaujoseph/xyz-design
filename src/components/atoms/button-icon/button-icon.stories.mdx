## Sizing

A button icon supports multiple sizes:

- default
- medium
- small
- x-small
- xx-small

Omit the `size` property for a button icon with the default size.

<Preview withToolbar>
  <Story name="sizing">

In xyz design we provide 2 types of button:

- neutral
- dropdown

- `disabled`
- `outline`
