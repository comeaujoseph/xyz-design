# Button Group

Button group should be used to group together buttons that are performing similar actions or acting on the same object.

<Preview withToolbar>
  <Story name="group of buttons" parameters={{ decorators: [withKnobs] }}>
    <ButtonGroup>
      <Button isOutline>edit</Button>
      <Button isOutline>save</Button>
      <Button isOutline>delete</Button>
    </ButtonGroup>
  </Story>
</Preview>
